How to Use Bulk Upload
Step 1: Click Export to Excel first to download the current Pakistan inventory as a CSV file.
Step 2: Open the CSV in Excel or Google Sheets. Edit the values you want to change. Do not change the Item Name column — it is used to match rows during upload.
What you can edit:
Column What it does
Qty in PakistanUpdates the quantity currently in Pakistan warehouse
Target ATL StockSets the target quantity for Atlanta — used for color-coded status
Shipment columnsUpdates qty for each active shipment on water — column header shows shipment name and ETA
Hot ItemType YES to mark as hot item, leave blank or type NO to unmark
NotesUpdates the notes field for that item
Important: Only rows with changed values will be updated. You will see a preview of all detected changes before anything is saved — review carefully before confirming. The ATL Qty column is read-only and cannot be changed via upload.
Do NOT change these columns:
Item Name, ProClean Name, UOM, Total On Water, Atlanta (ATL) Qty — these are read-only. Changing Item Name will cause rows to not be found during upload.
