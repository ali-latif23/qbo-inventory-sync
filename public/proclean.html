How to Use Bulk Upload
Step 1: Click Export to Excel first to download the current inventory as a CSV file.
Step 2: Open the CSV in Excel or Google Sheets. Edit the values you want to change. Do not change the Item Name or SKU columns — those are used to identify which items to update.
What you can edit:
Column What it does Updates QBO?
Qty On HandUpdates the current stock levelYes — ProClean QBO
Sale PriceUpdates the selling priceYes — ProClean QBO
Cost PriceUpdates the purchase costYes — ProClean QBO
UOMUpdates how the unit is displayed on the website onlyWebsite only — not QBO
Monthly ConsumptionSets the target stock level shown next to quantity, used to calculate Months of Stock on the Pakistan tabWebsite only — not QBO
Sales DescriptionUpdates the sales descriptionYes — all 3 companies
Important: Only rows with changed values will be updated. Items with no changes are automatically skipped. You will see a preview of all detected changes before anything is saved — review carefully before confirming.
Do NOT change these columns:
Item Name, SKU, Sale Value, Cost Value, Last Synced, Last Updated By — these are read-only reference columns. Changing them has no effect on the upload (except Item Name which is used for matching — do not change that).
